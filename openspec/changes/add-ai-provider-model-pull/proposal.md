# 后台拉取供应商模型列表

## Purpose

管理员新增 AI 模型时目前只能手动输入模型名，容易填错，也不知道供应商当前有哪些可用模型。需要在模型表单中提供“拉取模型”能力，按已配置的供应商读取模型列表，让管理员直接选择。

## Scope

- 后台 AI 模型表单在模型名旁增加拉取按钮。
- API 增加只读接口，按供应商拉取模型列表。
- OpenAI-compatible 适配器支持请求供应商 `/models`。
- 选择或输入模型名时，展示名默认使用“用途名称 + 模型名”。

## Non-Goals

- 不新增 AI 供应商类型。
- 不保存供应商返回的完整原始模型元数据。
- 不新增数据库字段或迁移。
- 不在响应中返回 API Key 或敏感配置值。
